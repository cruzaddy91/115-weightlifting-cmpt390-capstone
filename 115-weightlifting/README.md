# 115 Weightlifting Source

This directory contains the application source used by the Docker Compose stack in the repository root.

For professor review, use the root-level workflow:

```bash
cd ..
cp .env.example .env
docker compose up --build
```

Then open:

- Frontend: `http://localhost:4173`
- Backend API: `http://localhost:8000`

The root `README.md` contains the full launch instructions, demo credentials, and smoke/stress validation command.
